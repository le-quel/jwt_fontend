CÁCH HẠ VERSION NODEJS

câu lệnh cài đặt các phiên bản nodejs "nvm install 14"
câu lệnh sử dụng "nvm use 14"
Hoặc nếu bạn muốn phiên bản Node.js này được sử dụng mặc định, bạn có thể chạy:
nvm alias default 14
